Análise Preditiva de Carteira de Capital Social

Cooperativa de Crédito Código indisponível por conter regra de negócio ou dados sensíveis.

Desenvolvi um sistema para avaliar possíveis cenários futuros de acordo com diferentes regras para saque de capital social para associados de cooperativa de crédito. O sistema simulou o comportamento da carteira de capital social a partir de dados dos últimos 20 anos. As simulações seguiram diferentes regras para percentual de saque por parcela, critérios para enquadramento na regra de saque e como a carteira se comportaria ao longo do tempo. Os resultados foram cruciais para a tomada de decisão sobre a alteração de regra que passou a vigorar a partir de 2023.

Python · R · Oracle · Pandas · Numpy · Matplotlib · Scikit-learn

Gerenciamento de Casos de COVID-19

Cooperativa de Saúde Código indisponível por conter regra de negócio ou dados sensíveis.

Desenvolvi um sistema para catalogar casos relacionados à COVID-19, utilizado via web em 8 postos avançados de atendimento, distribuídos em 5 municípios e utilizado por mais de 30 colaboradores. O sistema coleta dados das bases de dados da cooperativa, buscando clientes que realizaram exames relacionados à detecção ou tratamento de sintomas do vírus e lista esses clientes para contato da equipe especializada. O módulo de monitoramento de positivados mantém os cadastros com dados gerais dos clientes e permite que a equipe de enfermeiros insira informações de contatos e relatórios sobre a evolução clínica. O sistema monitora os dias de quarentena e gera alertas de acompanhamento para a equipe. As funções do sistema incluem a gestão de casos suspeitos, positivados, negativados, reincidentes e óbitos, além de cadastros, manutenção e emissão de relatórios.

Python · Oracle · Django · SQLite

Monitor de Processos da Equipe de BI

Cooperativa de Crédito

Desenvolvi um sistema web de gerenciamento de processos para monitorar todas as rotinas de atualizações de bases de dados e dashboards da cooperativa. O processo antes era realizado manualmente por 3 colaboradores que precisavam validar a atualização dos dashboards e, em caso de erro, executar as rotinas manualmente. No sistema, todas as rotinas desenvolvidas pela equipe são cadastradas, definindo periodicidade (diária, semanal, mensal, anual ou sob demanda), linguagem de desenvolvimento e descrições gerais sobre o processo. O módulo de agendamento permite definir quando a rotina será executada, de acordo com a periodicidade de atualização. O painel de atualizações permite filtrar e visualizar todos os status de execução das rotinas, validando com a periodicidade de cada processo, listando logs de execução e permitindo, em casos de falha, visualizar o erro e executar novamente o processo dentro do sistema, permitindo que integrantes da equipe sem conhecimento de programação interajam com os processos.

Python · Oracle · R · Pandas · Numpy · Subprocess · SQLite · JavaScript

Construtor de HTML Dinâmico

Cooperativa de Crédito

Construí um sistema de disparos de e-mails automatizados, que pode receber qualquer base de dados que contenha informações que possam identificar a qual carteira ele pertence. O sistema recolhe a base de dados e, a partir de um HTML base com as colunas criadas, gera um arquivo por gerente de negócio, preenchendo os campos da tabela apenas com informações referentes aos seus associados. Após criar dinamicamente a tabela, o sistema exporta em um diretório e preenche um arquivo CSV com as informações de destinatário, pessoas em cópia e o anexo do arquivo em HTML que preenche o corpo do e-mail. Este sistema faz parte de duas rotinas que disparam mensalmente mais de 2500 e-mails populados com tabelas em HTML criadas dinamicamente. O processo para construção e envio dura de 2 a 4 segundos.

Python · R · BeautifulSoup · Pandas · Babel

Validador de Situação do CNPJ na Receita Federal

Cooperativa de Saúde

Desenvolvi um sistema que coleta os dados de clientes na base de dados da cooperativa de saúde e valida através da API da ReceitaWS, buscando validar os dados de situação cadastral. Previamente, o processo era realizado por uma colaboradora que buscava manualmente a partir da data de aniversário do contrato e validava a situação do CNPJ, evitando casos onde o contrato persistia vigente após o encerramento do CNPJ. O sistema foi desenvolvido para operar de maneira gratuita de acordo com as regras da API. O sistema opera em tempo integral, alertando via e-mail quando ocorre algum caso de inconsistência, validando mais de 60 mil CNPJs da base de dados.

Python · Requests · Pandas
